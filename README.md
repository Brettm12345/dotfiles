## Instalation
Set `DOTFILES` to whatever you want.
I personally use `~/etc`
``` shell
export DOTFILES="${HOME}/etc"
git clone git://github.com/Brettm12345/dotfiles "$DOTFILES"
eval "$DOTFILES"/install.sh
```
